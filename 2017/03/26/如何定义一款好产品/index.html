如何定义一款好产品

PM马尔 3月 26, 2017
  • 产品
  • 分享到微博
  • 分享到 QQ

1、对企业来说，它吸引用户（被需要）;它产生利润（及时有效地满足用户需求）。
对用户来说，忽视不了它的存在（需要它）；感受不到它的压迫（喜欢它）。

2、是否能够赢得用户时间。
相关联的是时间的投入产出分别是什么。就时间投入而言，产出的定义发生变化，比如听歌发呆，这样的产出是一份难得的闲适。
如果需要有产出，那么就将这种产出需要被放大到极致，发挥到最优雅。

3、从需求来说，好产品真正地解决了某个具有普遍性的问题。从体验来说，好产品让人感觉有效、流畅。

4、从表现来说，好产品符合某个统一的设计规范，软件系统内的各个元素都不存在随意而为的情况。从服务来说，好产品就意味着好服务，好的服务让人感到满意。要让人感到满意，就要适时、好玩、满足、低成本。

5、从质量来说，好产品必然具有很好的稳定性，对异常情况做到了最大程度地友好处理。

6、从效果来说，好产品可以提升用户的幸福指数。好产品可以提升社会的运转效率。
